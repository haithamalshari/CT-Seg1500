# CT-Seg1500
CT-Seg1500: A harmonized, slice-thickness-aware, transformer-ready multi-source brain CT cohort and preprocessing framework for intracranial hemorrhage segmentation and diagnosis. Includes curation/QC reports, conversion &amp; precompute pipelines, and reproducible build logs for NIfTI (.nii.gz), raw NumPy, and precomputed multi-window NumPy formats.
